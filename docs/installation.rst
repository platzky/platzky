Installation
============

Python Version
--------------

Platzky requires Python 3.10 or higher.

Installing Platzky
------------------

Install Platzky using pip:

.. code-block:: bash

    $ pip install platzky

Or with Poetry:

.. code-block:: bash

    $ poetry add platzky

Verify Installation
-------------------

To verify that Platzky is installed correctly:

.. code-block:: python

    >>> import platzky
    >>> print(platzky.__version__)
    |version|
